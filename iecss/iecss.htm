<h1>
Internet Explorer User Agent Style Sheets
</h1>
<p>
The <abbr title="User Agent">UA</abbr> Style Sheet is a simple set of css styles that each web browser uses before any other css styles are applied.
</p>
<p>
This chart lists and compares the different default style sheets used to render HTML in the four major versions of Internet Explorer; <abbr title="Internet Explorer 6">IE6</abbr>, <abbr title="Internet Explorer 7">IE7</abbr>, <abbr title="Internet Explorer 8">IE8</abbr>, and <abbr title="Internet Explorer 9">IE9</abbr> Platform Preview.
</p>
<p>
You can download each of these <abbr title="User Agent">UA</abbr> stylesheets by using the links at the top of this chart.
